# QuizApp

## Beschreibung
Die QuizApp ist eine kleine interaktive Webanwendung, bei der Nutzer Fragen beantworten können.

Das Projekt demonstriert die Umsetzung von Logik und Benutzerinteraktion mit JavaScript.

## Funktionen
- Anzeige von Quizfragen
- Auswahl von Antwortmöglichkeiten
- Auswertung der Antworten
- Dynamische Rückmeldung an den Nutzer

## Verwendete Technologien
- HTML5
- CSS3
- JavaScript

## Lernziele
- Steuerung von Programmabläufen
- Arbeiten mit Bedingungen und Variablen
- Benutzerinteraktion und Feedback
- Dynamische Aktualisierung der Benutzeroberfläche

## Projekt starten
Die `index.html` Datei im Browser öffnen.
